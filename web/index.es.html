¿Qué hace este panel exactamente?

Principio de funcionamiento

El juego oculta las opciones de ray tracing según una lista blanca de modelos de GPU. OptiScaler modifica la información de GPU devuelta por DXGI/Streamline dentro del proceso del juego, haciéndole ver el perfil que hayas elegido (tu GPU real, RTX 4090 o RTX 5080M).

Diferencia con modificar el registro

Modificar el registro cambia el nombre de la GPU para todo el sistema. Esta herramienta solo reemplaza el DLL proxy local en el directorio del juego, sin tocar el registro de la GPU.

Estrategia de respaldo

Antes de instalar se guardan winmm.dll, OptiScaler.ini, OptiScaler\ y archivos antiguos de DLSSTweaks, con un manifest detallado.

Qué hacer si falla

Si el juego no arranca, restaura el respaldo más reciente. Si la opción no aparece, revisa el registro para ver si OptiScaler se cargó correctamente.

Proyectos principales

El núcleo de GPU spoof proviene de OptiScaler. DLSSTweaks solo se menciona por compatibilidad con archivos antiguos; esta herramienta no usa su flujo de escalado DLSS.
