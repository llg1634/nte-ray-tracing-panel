这个面板到底做了什么

成功原理

游戏通过 GPU 型号白名单隐藏光追选项。OptiScaler 在游戏进程内改写 DXGI/Streamline 返回的 GPU 信息，让它看到你选择的本机、4090 或 5080M profile。

和注册表改名的区别

注册表改名是整机级影响；本工具默认只替换游戏目录本地代理 DLL，不改系统显卡注册表。

备份策略

安装前保存 `winmm.dll`、`OptiScaler.ini`、`OptiScaler\` 和旧 DLSSTweaks 文件，并写入 manifest。

失败处理

如果游戏无法启动，先恢复最近备份；如果选项没出现，再查看日志是否加载了 OptiScaler。

核心项目

GPU spoof 核心来自 OptiScaler。 DLSSTweaks 仅作为旧本地文件兼容备份说明，本工具不使用它的 DLSS 比例流程。
